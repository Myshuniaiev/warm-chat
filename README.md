## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Myshuniaiev/warm-chat.git
   ```

2. Install dependencies:
   ```bash
   yarn install
   ```

3. Configure the PostgreSQL database connection in the .env file.

4. Start the development servers:
   ```bash
   yarn start
   ```
